# Grade Manager

## What is this?
Grade Manager is a command-line tool that helps you easily 
track and manage student grades.

## Features
- Add students with their grades
- View the full list of students
- View statistics: average, highest, and lowest grade
- Save students to a file
- Load students from a file automatically on startup
